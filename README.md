# Movit Energy — Market Size Estimator

Estimates competitor market size for any energy/supplement product across CZ, EU, and USA.

## Quick Start

### 1. API Key
Get a free Serper.dev key at https://serper.dev (2500 free searches/month).

### 2. Backend
```
cd backend
cp ../.env.example .env       # Fill in your SERPER_API_KEY
pip install -r requirements.txt
python main.py
```

### 3. Frontend
```
cd frontend
npm install
npm run dev
```

Open http://localhost:5173
